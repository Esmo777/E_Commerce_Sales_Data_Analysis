Overview

This project demonstrates a complete end-to-end data analytics workflow, from raw data to business insights and visualization.
The objective is to analyze sales performance, identify trends, and deliver actionable insights using Python, SQL, and Power BI.

The project covers:

- Data loading and preprocessing in Python
- Exploratory Data Analysis (EDA)
- Data cleaning and transformation
- SQL analysis using MySQL Workbench
- Interactive dashboard creation in Power BI
- Business reporting and presentation (Gamma PPT)

Dataset

- Type: E-commerce sales data
- Content:
  . Product information
  . Categories
  . Sales, profit, and time variables
- Granularity: Monthly sales records
- Purpose: Analyze sales performance, profitability, and product trends

Tools & Technologies
- Python: Pandas
- SQL: MySQL Workbench
- BI Tool: Power BI
- Presentation: Gamma (AI-generated PPT)
- Environment: Jupyter Notebook

Project Steps
1. Data Loading (Python)
 - Imported dataset using Pandas
 - Checked data structure, data types, and size

2. Exploratory Data Analysis (EDA)
 - Summary statistics
 - Sales and profit distribution analysis
 - Trend analysis by product, category, and time
 - Detection of missing values and outliers

3. Data Cleaning & Preparation
 - Handled missing values
 - Standardized column names
 - Corrected data types
 - Created derived metrics (aggregated KPIs, ...)

4. SQL Analysis (MySQL Workbench)
- Loaded cleaned data into MySQL
- Wrote SQL queries to:
  . Calculate total sales and profit
  . Identify top 10 products
  . Analyze performance by category and month
  . Support dashboard metrics

5. Power BI Dashboard
- Built an interactive dashboard including:
  . Total sales & profit KPIs
  . Top 10 products
  . Category performance
- Added filters for dynamic exploration

6. Reporting & Presentation
- Created a structured analytical report
- Designed a professional presentation using Gamma
- Focused on insights, not just visuals

Dashboard
The Power BI dashboard provides:
- Clear KPIs for decision-makers
- Visual comparison of products and categories
- Time-based trend analysis
- Interactive filtering for deeper insights

Key Results & Insights
- Identified top-performing products by sales
- Highlighted most profitable categories
- Detected seasonal sales patterns
- Provided data-driven recommendations for business growth

How to Run the Project

1.Clone the repository
git clone https://github.com/your-username/data-analytics-project.git

2. Install dependencies
pip install pandas numpy sqlalchemy pymysql

3. Run the Python notebook
- Open notebooks/analysis.ipynb
- Execute all cells in order

4. SQL Analysis
- Import the cleaned dataset into MySQL
- Run SQL scripts from the /sql folder using MySQL Workbench

5. Power BI
- Open the .pbix file in Power BI Desktop
- Refresh data connection if needed
